# 🎬 Movie Recommendation System

Hi! This is a **Movie Recommendation System** project developed using **Python and Machine Learning techniques**.

## 📌 Project Overview

This application allows users to select a movie they like and instantly receive **five similar movie recommendations** along with their posters. It provides a simple and interactive way to discover new movies based on personal preferences.

## ⚙️ How the System Works

The project is built using the **TMDB 5000 Movies dataset**. A **content-based recommendation approach** is used where movie similarity is calculated through **cosine similarity**.

The web interface is developed using **Streamlit**, and movie posters are dynamically fetched through the **TMDB API**.

## 📂 Project Files

The repository includes the following important files:

* `app.py` – Main Streamlit application that runs the web interface
* `movies.pkl` – Processed dataset containing movie information
* `movies_dict.pkl` – Dictionary version of the movies dataset
* `similarity.pkl` – Precomputed similarity matrix used for recommendations

## 🛠 Technologies Used

This project was created using the following technologies:

* Python
* Pandas
* Scikit-learn
* Streamlit
* TMDB API
